- Google Cloud平台基本功能操作
- AI和机器学习服务介绍
- 数据质量提升与探索性数据分析
- 使用Vertex AI AutoML构建、训练和部署机器学习模型
- TensorFlow和Keras模型设计
- 机器学习模型准确性提升
- 机器学习模型规模化使用
- 特征工程
- 企业级机器学习工作流程
- 生产级机器学习系统实施
- 计算机视觉基础
- 自然语言处理
- 推荐系统构建
- 机器学习运营(MLOps)入门
- MLOps工具与实践

- 在Vertex AI上构建和部署机器学习解决方案
